## Welcome to Chorus

Chorus is a Voice AI platform that enables you to build and deploy intelligent conversational agents for phone calls. Create agents that can handle customer support, sales calls, lead qualification, and more - all through a simple REST API and intuitive dashboard.

## Key Features

- **Voice AI Agents**: Build intelligent conversational agents with customizable prompts and behaviors
- **Multi-Provider Support**: Use Twilio or SIP trunking for telephony
- **Custom Tools**: Create HTTP tools that agents can call to fetch data or perform actions
- **Real-time Conversations**: Low-latency voice interactions powered by state-of-the-art AI models
- **Call Management**: Track calls, view transcripts, and access call recordings
- **Flexible Configuration**: Customize voices, models, greeting messages, and conversation flows
- **Post-Call Webhooks**: Get notified when calls complete with summaries and extracted data
