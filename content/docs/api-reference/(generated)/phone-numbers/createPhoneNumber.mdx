Register a new phone number. For Twilio, provide account credentials. For SIP, choose between 'registration' mode (our SIP server registers TO your PBX) or 'ip_allowlist' mode (your PBX IP is allowlisted). SIP numbers support outbound proxy and auth user options in registration mode.
